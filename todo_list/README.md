# To-Do List (Python CLI)

## Deskripsi
Aplikasi sederhana untuk mencatat tugas harian berbasis command line.

## Fitur
- Tambah tugas
- Lihat daftar tugas
- Tandai tugas selesai
- Hapus tugas
- Data tersimpan otomatis (JSON)

## Cara Menjalankan
1. Install Python
2. Jalankan:
   python main.py

## Teknologi
- Python
- JSON
